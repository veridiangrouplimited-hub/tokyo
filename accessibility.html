Accessibility Statement

Our commitment to making this website accessible to everyone.

  1. Home
  2. Accessibility Statement
Last Updated: 1 June 2026Applies to: nigerianembassy.[xx]Standard: WCAG 2.1 AA

The Embassy of the Federal Republic of Nigeria in Japan is committed to ensuring that this website is accessible to all users, including those with disabilities. We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standard and continually work to improve the accessibility of our digital services.

1. Our Commitment

We believe that everyone has the right to access information about consular services, public notices, and Mission activities. We are working to ensure that this website can be used by people who:

  • Use a screen reader or other assistive technology
  • Navigate by keyboard rather than a mouse
  • Use voice input software
  • Have low vision or colour blindness
  • Have cognitive or learning disabilities
  • Access the website on a mobile device or small screen

2. Technical Specifications

This website has been designed and developed to be accessible by incorporating the following features:

Structure and Navigation

  • Semantic HTML5 elements (header, nav, main, footer, article, section, aside) are used throughout
  • A "Skip to main content" link is provided at the top of every page for keyboard users
  • Breadcrumb navigation is provided on all inner pages
  • All pages have a clear and descriptive <title>
  • Heading levels are used in correct hierarchical order (H1 → H2 → H3)
  • Navigation menus are keyboard-accessible with full arrow-key support

Colour and Contrast

  • Text and background colours meet or exceed WCAG 2.1 AA contrast ratio requirements (4.5:1 for normal text, 3:1 for large text)
  • Colour is never used as the sole means of conveying information
  • Focus indicators are clearly visible for all interactive elements

Images and Media

  • All meaningful images have descriptive alt text attributes
  • Decorative images use empty alt="" attributes so screen readers skip them
  • Icons used alongside text are marked aria-hidden="true"

Forms

  • All form fields have associated <label> elements
  • Required fields are clearly indicated both visually and programmatically
  • Error messages are descriptive and linked to the relevant field
  • Forms can be completed and submitted using keyboard only
  • Form controls have visible focus states

Keyboard and Focus

  • All interactive elements (links, buttons, form controls) are reachable and operable by keyboard
  • Dropdown menus support Escape key to close and arrow keys to navigate
  • Focus is never trapped on an element without a way to escape
  • Tab order follows the visual reading order of the page

Responsive Design

  • The website is fully responsive and works correctly on desktop, tablet, and mobile devices
  • Text can be resized up to 200% without loss of content or functionality
  • Content does not require horizontal scrolling on standard viewports
  • Touch targets are large enough for comfortable use on mobile devices

3. Known Accessibility Limitations

We are aware of the following limitations and are working to address them:

  • Some older PDF documents linked from the website may not be fully accessible. We are working to replace these with accessible versions.
  • Embedded maps (where applicable) may not be fully accessible to screen reader users. Alternative text descriptions of the Mission's location are provided.
  • Some video content, where published, may not yet have captions or audio descriptions. We are implementing a captioning policy for all new video content.

4. Assistive Technology Compatibility

This website has been tested with the following assistive technologies:

  • NVDA (NonVisual Desktop Access) with Firefox
  • JAWS (Job Access With Speech) with Chrome
  • VoiceOver (macOS and iOS) with Safari
  • TalkBack (Android) with Chrome
  • Windows High Contrast Mode
  • Keyboard-only navigation (no mouse)

5. Browser Compatibility

This website is designed to work correctly with all modern browsers, including:

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Microsoft Edge (latest version)
  • Apple Safari (latest version)

Internet Explorer is not supported. We recommend using a modern, up-to-date browser for the best experience.

6. Reporting Accessibility Issues

We welcome feedback on the accessibility of this website. If you encounter an accessibility barrier or have difficulty accessing any content, please contact us:

  • Email: info.tokyo@foreignaffairs.gov.ng
  • Telephone: +[Code] [Number] (Mon–Fri, 9:00 AM – 4:00 PM)
  • Post: Accessibility Officer, Embassy of Nigeria, [Address], Tokyo, Japan

Please provide as much detail as possible, including the page you were visiting, the problem you encountered, and the assistive technology or browser you were using. We aim to respond within 10 working days and to resolve reported issues as quickly as possible.

7. Formal Complaints

If you are not satisfied with our response to your accessibility concern, you may escalate the matter to the appropriate authority in Japan responsible for public sector digital accessibility.

8. Review and Updates

This Accessibility Statement is reviewed annually and updated to reflect any improvements made to the website. We commission accessibility audits periodically to identify and address areas for improvement.

♿

We are committed to continuous improvement of the accessibility of this website. If there is anything we can do better, please contact us.
